# AiON Phase 1: Core Engine

Welcome to AiON! This is your dream project, and we've set up the foundation for you.

## How it works
This code uses AI agents (Planner, Architect, Coder) connected through a LangGraph State Machine. They pass information to each other to build out your software project automatically!

## Setup Instructions

1. **Install Python**: Make sure you have Python installed on your computer.
2. **Open Terminal**: Open your command prompt or terminal in this folder (`aion-phase1`).
3. **Install Dependencies**: Run the following command to install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```
4. **Set API Key**: Open the `.env` file and replace `your_gemini_api_key_here` with your actual Gemini API key.
5. **Run the Project**:
   ```bash
   python backend/main.py "Build a student management system"
   ```

Watch the magic happen!
